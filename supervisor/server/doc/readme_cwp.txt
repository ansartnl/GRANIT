Для установки SupervisorServer сервиса нужно использовать install.bat
Для деинсталяции сервиса SupervisorServer - uninstall.bat
Запуск сервиса можно выполнить с использованием start.bat, restart.bat или с помощью
консоли "Servce Control Manager"
Остановка сервиса выполняется или с использованием stop.bat или с помощью "Servce Control Manager"

Описанией файла конфигурации:
1. Файл конфигурации содержит 3 секции и 2 параметра.
   Для рабочего места CWP и FDP файлы конфигурации отличаются.
   Секции:
   - fmtp_services:
     указываются FMTP сервисы и порядок их запуска: имя тега - FMTP сервис, значение - порядок запуска
   - stca_service:
     значение этого параметра - название STCA сервиса (stca)
   - cwp:
     path - абсолютный путь к исполняемому файлу CWP.exe
     arguments - командная строка cwp.exe (указывается абсолютный путь к конфигурационному файлу
     CWP в кавычка(кавычки нужны для корректной обработки путей с пробелами))
     user - имя пользователя (для автоматической авторизации)
     password - пароль (для автоматической авторизации)
   - database - параметры соединения с базой данных, запуск которой будет валидироваться
   - language - english или russian
   - log - состояние логирования. Когда значение true - выполнять логирование, иначе - не выполнять
2. Файл конфигурации может находится:
   - в текущей папке с сервисом и должен имет имя сервиса SupervisorServer.xml
   - в папке ../etc/config.xml
   - передаваться первым аргументом в командной строке

Организация работы сервиса:
- Выполняется попытка соединения с базой данных до тех пор пока не будет установлено соединение
- Запускаются FMTP сервисы в порядке указанном в конфигурации
- Запускается STCA сервис
- Запускается CWP.exe
Если в процессе работы возникает критическая ситуация, например FMTP сервисы перестают работать
SupervisorServer в течении 5 сек выполняет перезагрузку сервисов и таким образом восстанавливает
их работоспособность.
Если CWP перестает работать - приложение тот час же перезапусается, таким образом обеспечивая 
непрерывное отображение аэронавигационной обстановки.
